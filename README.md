# 8-PUZZLE
O sistema implementa o clássico problema do 8-Puzzle, permitindo a resolução do jogo por meio de algoritmos de busca, com diferentes níveis de dificuldade.  O objetivo é aplicar conceitos de estruturas de dados, manipulação de estados, busca em espaços de estados e lógica computacional, utilizando a linguagem C.
